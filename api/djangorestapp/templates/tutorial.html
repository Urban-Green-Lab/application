How to get Started

  1. Add questions to the question bank
  2. Create quizzes from the questions
  3. Schedule events with a quiz
  4. To download a csv of players, click on View Quiz Takers
    • List can be filtered by event on the right
    • Click select or select all rows of what information you want to download
    • Then in the action dropdown select "Export Selected As CSV" and click go
